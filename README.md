# Gayola Punk Rock – Official Website

## Descripción
Sitio web corporativo desarrollado para una banda musical, enfocado en rendimiento, estructura semántica y diseño responsive.

## Stack
- Vue 3 + Vite
- CSS personalizado
- Arquitectura basada en componentes
- PWA (VitePWA)

## Decisiones Técnicas
- Enfoque mobile-first
- Componentes reutilizables
- HTML5 semántico
- Optimización manual de imágenes (WebP + compresión)
- Auditoría de rendimiento con Lighthouse

## Rendimiento
Auditoría Lighthouse centrada en:
- Performance
- Accessibility
- Best Practices
- SEO

## Instalación
npm install
npm run dev
